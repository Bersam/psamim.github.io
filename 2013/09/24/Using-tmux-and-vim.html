Quickly open a tmux pane in Vim

Using terminal is essential in Vim for me. There are some shells for vim but they are slow, blocking or lack features. With tmux you have your own real shell beside you. The first thing I wanted to do was to be able to quickly open a new pane, this tmux script does that:

splitw -v -p 20           # Split pane 1 horizontally by 20%

I saved this in a file calld vimo and set a key mapping for that:

bind t source-file ~/.tmux/vimo
bind q kill-pane -a # Close all other panes but this
